# Satwise Website

A static landing page built from the content of satwise.me.

## Features

- Bitcoin consulting service sections
- Download and education links
- Appointment booking calls to action
- Contact form layout and business hours
- Blog preview section

## Usage

1. Open `index.html` in your browser.
2. Or use a local static server such as `python3 -m http.server`.

## Notes

- The site is built as a simple static HTML/CSS/JS project.
- Email links use `mailto:info@satwise.me` for direct contact.
